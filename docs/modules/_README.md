# Module Quick Reference

Condensed overviews of core SAM AI modules. For detailed documentation, see [04_modules/](../04_modules/).

## Modules

- **ai-brain** - Central AI processing and decision engine
- **ai-sam** - Main chat interface and user interaction
- **ai-sam-intelligence** - Context awareness and learning
- **ai-sam-lead-generator** - Lead capture and qualification
- **ai-sam-memory** - Conversation history and recall
- **ai-sam-workflows** - N8N-style automation builder
