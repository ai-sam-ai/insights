### Frequently Asked Questions

**Q: Do visitors need to create an account?**
A: No. The chat works without login. Session IDs are stored in the browser.

**Q: Can I customize the appearance?**
A: The widget uses Claude-style theming by default. Custom theming requires CSS overrides.

**Q: What happens if SAM doesn't know the answer?**
A: SAM routes complex questions to appropriate specialist agents or suggests contacting support.
